title: Boston Dynamics делает форк ядра Linux - Ядро Linux - Новости

description:
Boston Dynamics, дочерняя компания Google, специализирующаяся на робототехнике, создает ответвление ядра Linux ориентированное на быстродействующий искусственный интеллект. Ядро будет оптимизировано для создания интегрированных нейросистем для использования в военной робототехнике, авиации, ракетном вооружении.
