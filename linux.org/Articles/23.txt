title: Релиз Blender 2.74 - Мультимедиа - Новости

description:
После выпуска четырёх релиз-кандидатов Blender Foundation рада наконец представить релиз Blender 2.74! В движке рендеринга Cycles: Оптимизирован чёрный фон мира. Оптимирирован нод Vector Math без входящего соединения со значением другого нода. Снижена задержка между изменением дерева шейдеров и рендером в рабочем окне. Различные оптимизации использования памяти. Добавлен атрибут, указывающий степень заострённости геометрии, доступный через одноимённый выход нода Geometry. Добавлена поддержка переноса текстурных координат с другого объекта. Нод Image Texture теперь поддерживает сферическую и цилиндрическую проекции текстур. Движок теперь поддерживает плоское отсечение вместо сферического. Добавлены настройки поля зрения для панорамной камеры. Исправлен старый баг с нодом Mapping, который давал результат сильно отличающийся от такового в Blender Internal renderer и режиме отображения Material. Это привело к поломке совместимости (проект, открытый в старых версиях, будет давать неправильный результат). Улучшения в моделировании: Кастомные нормали, то есть нормали, отличные от автоматически рассчитанных. Основные возможности реализованы в виде Python-обвязки для импорта нормалей (например, из FBX), возможности передавать нормали через инструменты передачи данных и в виде модификалора Set Split Normals Инструмент Mesh Data Transfer - в виде оператора и модификатора. Передаёт данные о сетке с одного объекта на другой, используя при этом различные методы разметки: пока что поддерживаются только UV, Vertex Color и Vertex Group. Этот инструмент позволяет редактировать кастомные нормали, описанные в предыдущем пункте. Модификатор Hook получил различные варианты спада воздействия. Vertex Slide теперь поддерживает корректировку UV. Добавлено обратно квадратичное смешение для режима пропорционального редактирования. Выбор одинаковых полигонов теперь имеет выборку по сглаженному/плоскому затенению. Анимация: Штрихи Grease Pencil можно копировать и вставлять между разными слоями, кадрами и даже блоками данных. С помошью Alt+S можно регулировать толщину штрихов. Хоткей L даёт свою обычную функциональность: выделяет соединённые элементы. Множество других улучшений. Изменения в интерфейсе: Композитинг рабочего окна. Пока что поддерживаются только два эффекта: Ambient Occlusion и Depth of Field. Outliner получил режим отображения осиротевших блоков данных (т. е. тех, которые ничем не используются или им назначен фальшивый пользователь). Возможность управлять вышеназванными блоками - сохранять их в проекте или нет. Ранее блоки, имеющие 0 пользователей, удалялись из проекта при сохранении и перезапуске. Также можно очистить проект от осиротевших блоков. Просмотрщик файлов получил поле поиска. Закладки в просмотрщике файлов теперь можно редактировать. Если модифицирующая клавиша не нажата, то нажатие ПКМ открывает контекстное меню. Система частиц: Исправление давней проблемы с соотношением между самыми длинными и самыми короткими волосами, которая приводила к «взрыву на макаронной фабрике». Волосы наконец-то поддерживают столкновения с меш-объектами. Объёмные расчёты. Теперь симулируется поведение не прядей, а объёмов (наподобие симуляции жидкости), что более эффективно. Изменена силовая модель для жёсткости волос. Модификаторы для управления потомками волос. Кнопки переключения модификаторов системы частиц вынесены в список систем частиц, чтобы не переключаться между разделами частиц и модификаторов. Оператор копирования систем частиц с одного объекта на другой. Новый инструмент причёсывания Shape Cut позволяет использовать форму меш-объекта для создания «стрижки».
